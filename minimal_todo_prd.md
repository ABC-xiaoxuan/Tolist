# 极简待办（MinimalTodo for Windows）PRD

## 1. 产品概述

极简待办是一款面向 Windows 平台的桌面待办应用，核心目标是降低用户记录、查看和执行每日计划的操作成本，让“规划每一天”这件事更直接、更轻松。

产品定位不是复杂的项目管理工具，而是 **轻量、清晰、专注** 的个人待办应用。除基础待办能力外，产品还提供：

- 今日待办管理
- 月视图查看一个月任务清单
- 浮窗快速操作
- 番茄钟
- 倒计时
- 自定义专注时长与倒计时长度

---

## 2. 产品目标

### 2.1 目标
- 让用户可以在较少步骤内完成任务添加、查看、完成和回顾
- 提供月视图，便于用户纵览一个月的待办安排
- 提供浮窗，减少在主窗口和其他应用之间来回切换的成本
- 提供番茄钟与倒计时，帮助用户在执行任务时保持专注
- 在保证功能完整的前提下，保持界面极简、信息清楚、视觉干净

### 2.2 产品原则
- **少干扰**：避免复杂层级和过多无关入口
- **强可读**：任务、时间、标签、状态要一眼看清
- **高频优先**：今日任务、快速添加、专注工具放在核心位置
- **轻量优先**：设计和实现都应避免无意义堆功能，降低资源占用风险

---

## 3. 目标用户

- 希望每天规划任务的个人用户
- 需要快速记录工作/学习/生活安排的人
- 使用番茄工作法或需要倒计时提醒的人
- 希望通过浮窗快速查看任务、不想频繁切主窗口的用户

---

## 4. 技术栈

- **桌面框架**：Electron
- **开发语言**：TypeScript
- **本地数据库**：SQLite
- **UI 实现**：自定义极简 UI
- **打包方式**：Electron Builder

> 说明：
> 技术选型以 Windows 桌面能力完整为前提，优先保证浮窗、托盘、快捷键、窗口管理和本地存储能力稳定，不盲目为了“更轻”牺牲关键桌面能力。

---

## 5. 核心功能范围

### 5.1 待办事项管理
- 添加任务
- 编辑任务
- 删除任务
- 标记完成/未完成
- 按日期查看任务
- 展示任务标签与任务时间
- 查看已完成任务

### 5.2 今日视图
- 展示当天待办列表
- 展示完成进度（如：已完成 3 / 8）
- 快速添加任务
- 对任务进行勾选、删除、编辑等操作

### 5.3 月视图
- 以日历形式展示整月任务分布
- 查看每天的任务数量和部分任务摘要
- 点击某一天，查看该日任务详情
- 展示本月完成率、任务统计、标签分布

### 5.4 专注工具
- 番茄钟
- 倒计时
- 自定义专注时长
- 自定义休息时长
- 自定义倒计时时长
- 开始、暂停、重置
- 计时结束提醒

### 5.5 浮窗功能
- 显示今日任务摘要
- 在浮窗中快速添加任务
- 在浮窗中勾选任务
- 在浮窗中查看番茄钟和倒计时
- 支持打开主界面
- 支持隐藏到托盘
- 支持置顶、收起、关闭

---

## 6. 主要使用场景

### 场景 1：快速规划今日任务
用户打开应用后，直接在“今天的待办”中添加任务，并查看今日完成进度。

### 场景 2：回顾整个月安排
用户进入“月视图”，查看本月每天的任务安排与完成情况，并点选某一天看详情。

### 场景 3：专注执行某项任务
用户在右侧专注工具区或浮窗中启动番茄钟/倒计时，配合待办执行任务。

### 场景 4：工作时使用浮窗
用户不想频繁切换到主窗口，只保留桌面浮窗，快速查看任务和启动计时。

## 8. UI 设计架构

### 8.1 整体风格
- Windows 11 风格，白色/浅灰背景
- 蓝色为核心强调色
- 卡片式布局，圆角 8~12px
- 弱阴影，减少视觉杂乱
- 信息分区明确
- 三栏布局：左侧导航，中间主内容，右侧专注工具

### 8.2 UI 配色（设计图为准）

#### 基础色板
| 用途 | 色值 | 说明 |
|---|---|---|
| 主强调蓝 | #4C8DFF | 主按钮、激活导航、进度条 |
| 页面背景 | #F6F8FC | 整体页面底色 |
| 卡片背景 | #FFFFFF | 任务卡片、专注卡片 |
| 边框灰 | #E5E7EB | 卡片边框、列表分隔线 |
| 主文字色 | #1F2937 | 标题、主要信息 |
| 次文字色 | #6B7280 | 说明文字、次级信息 |
| 完成任务背景 | #EEF5FF | 已完成任务行 |
| 成功/完成绿 | #22C55E | 完成状态反馈 |
| 标签色-工作 | #DBEAFE | 蓝色文本 |
| 标签色-设计 | #DCFCE7 | 绿色文本 |
| 标签色-学习 | #FEF3C7 | 黄色文本 |
| 标签色-健康 | #F3E8FF | 紫色文本 |

#### 功能状态色
| 状态 | 色值 | 用途 |
|---|---|---|
| 默认按钮 | #4C8DFF | 添加/开始/确认按钮 |
| 按钮悬停 | #3B82F6 | 鼠标悬停效果 |

### 8.3 模块分区

#### 主界面：三栏布局
- 顶部栏：应用名、日期、主题、设置、用户、窗口控制
- 左侧导航栏：今日、月视图、专注、已完成、设置 + 底部卡片
- 中间主内容区（今日待办）：标题、进度条、快速添加、任务列表、底部提示
- 右侧工具栏：番茄钟、倒计时、月统计卡片

#### 月视图布局
- 中间月历：月份导航、统计摘要、日期格显示任务点/小卡片、当前日期高亮
- 右侧详情：选中日期任务列表、本月完成率、标签分布、简单图表

#### 浮窗布局
- 浮窗头部：标题、置顶、折叠、关闭按钮
- 日期区：显示当天日期
- 快速添加区：输入框 + 添加按钮
- 今日任务摘要区：任务列表（压缩）
- 番茄钟区：25:00 + 控制按钮 + 自定义时长
- 倒计时区：自定义时间 + 控制按钮
- 底部操作区：打开主界面、隐藏到托盘、设置图标

## 9. 交互说明

- 点击复选框切换任务完成状态
- 点击任务更多操作进行编辑/删除
- 点击日期切换当日任务
- 月视图左右箭头切换月份
- 点击“今天”回到当前日期
- 专注工具：开始/暂停/重置番茄钟或倒计时
- 浮窗：全局快捷键呼出/隐藏，可拖拽移动、置顶、折叠、关闭

---

## 10. 数据存储

- SQLite 本地存储任务信息、日期、完成状态、标签、时间、番茄钟/倒计时配置、浮窗位置大小、主题设置
- 自动保存，无需手动操作
- 数据结构简洁，避免冗余设计

---

## 11. 非功能性需求

- 启动速度快，内存占用低（50~100MB）
- Windows 10/11 兼容
- 高频操作可快速响应，界面清晰
- 数据本地存储，保证隐私安全

---

## 12. 成功指标

- 用户可在 3 秒内完成添加任务
- 浮窗使用率 ≥50%
- 番茄钟/倒计时功能使用率 ≥40%
- UI 极简、信息清晰、操作直观
- 内存占用合理，不影响常规办公
